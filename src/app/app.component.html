Alert Component Library

Usage example:

In template:

<rpx-alert message="Lorem ipsum dolor sit amet, consectetur adipiscing elit" type="danger"> </rpx-alert>

In module where you use it:

import { RpxAlertModule } from '@rpx-components-ui/components';

and in "imports" section of @NgModule

Description:

Component to output rpx-components-ui styled alert of specified color(type)

Component parameters:
[type]
Alert type with possible options success | info | warning | danger. The default is danger.
[message]
Alert text to be displayed
[title]
Alert title
